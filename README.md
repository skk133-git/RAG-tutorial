# RAG-tutorial
Basic AI agent chatbot that uses groq api key. Allows users to upload documents and ask questions that will be answered by the ai.
Make sure to run in a virtual environment and import the langchain and vectorstore packages.

pip install flask
pip install langchain
pip install langchain-community
pip install langchain-core
pip install langchain-text-splitters
pip install faiss-cpu
pip install torch
pip install pypdf
pip install python-dotenv
pip install groq
